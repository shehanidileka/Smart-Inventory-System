

# Smart Inventory System 📦

## Overview

The Smart Inventory System is a robust web-based solution designed to automate and optimize inventory management for businesses. This system enables real-time stock tracking, automated reporting, and efficient data analysis to ensure seamless business operations.

## Key Features 🚀

* **Real-time Stock Tracking:** Monitor stock levels instantly with accurate data updates.
* **Automated Reporting:** Generate comprehensive daily and monthly analytical reports with ease.
* **User Management:** Secure, role-based access control for different user levels.
* **Intuitive Dashboard:** A clean, user-friendly interface for streamlined navigation.
* **Data Security:** Centralized and secure storage for sensitive inventory data.

## Technical Details 🛠️

* **Development IDE:** NetBeans
* **Backend:** Java Servlets, JSP
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Database:** MySQL (managed via phpMyAdmin)
* **Server:** Apache Tomcat 8.5

## Installation Guide ⚙️

Follow these steps to set up the project on your local machine:

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/smart-inventory-system.git

```


2. **Database Setup:**
* Open `phpMyAdmin`.
* Create a new database (e.g., `smart_inventory_db`).
* Import the provided `.sql` file located in the project folder.


3. **Configuration:**
* Open `DBConnection.java` within the project.
* Update your database credentials (URL, username, and password) accordingly.


4. **Running the Application:**
* Open the project in **NetBeans IDE**.
* Perform a **Clean and Build**.
* Run the project on **Apache Tomcat Server**.
